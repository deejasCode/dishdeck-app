# dishdeck-app
Android recipe management app built with Kotlin and Spring Boot

#DishDeck
A personal recipe management Android app that allows users to store, browse, and manage their recipes in one place

##Tech Stack
- Android (Kotlin)
- Spring Boot (Java) - REST API backend
- PostgreSQL - Database
- Docker & Docker Compose

##Architecture
C4 Container Model - Android app communicated with a Spring Boot REST API , which persists data in a PostgreSQL database.

##Screens
- Recipe List
- Recipe Detail
- Add / Edit Recipe
- Favourites
